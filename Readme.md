# FoxCraftCandide Support Center

This repository hosts the official support center for FoxCraftCandide applications. It serves as a central hub for users to find answers to frequently asked questions, contact support, and access important policy documents such as privacy policies and the CSAE policy.

## Features

- **Comprehensive FAQs:** A section dedicated to frequently asked questions to help users quickly resolve common issues.
- **Multiple Contact Options:** Users can easily reach support via email, GitHub issues, or WhatsApp.
- **Dynamic Privacy Policy Selection:** Provides specific privacy policies for various FoxCraftCandide applications, accessible via a selector.
- **CSAE Policy Information:** Clear access to the Child Sexual Abuse Exploitation (CSAE) policy.
- **App Reviewer Notes:** Dedicated section for app reviewers providing essential information for app store submissions.
- **Responsive Design:** Optimized for various devices to ensure accessibility.

## Applications Covered

The support center provides privacy policies for the following FoxCraftCandide applications:

[Privacy Policies](https://foxcraftcandide.github.io/support/index.html#privacy)

Users can select their specific application from a dropdown to view the relevant policy.
